# LearnPlaywirghtBatch

Quick notes to run the example scripts locally.

Prerequisite
- Install Node.js (LTS) on Windows: https://nodejs.org/

Run
```powershell
Push-Location 'C:\LearnPlaywirghtBatch\chapter_01_Basics'
npm run start   # runs 01_basic.js
npm run step    # runs 02_step_by_step.js
Pop-Location
```

If `node` is not recognized, make sure the Node installer added it to your PATH and restart your terminal.
